# Security Policy

## MySecurityMetrics / ComplianceQuest

---

## Supported Versions

This Product is currently in **private beta**.  
No public production releases are supported at this time.

---

## Reporting a Security Issue

We welcome **responsible, good-faith security disclosures**.
AGAIN we WELCOME **responsible, good-faith security disclosures**.

If you believe you have identified a security issue:

- **Stop testing immediately**
- **Do not exploit the issue**
- **Do not perform automated testing**
- **Do not attempt data access or exfiltration**
- **Do not disclose publicly**

## Prohibited Security Activity

The following are **not authorized**:

- Penetration testing
- Scanning or fuzzing
- Traffic interception
- Reverse engineering
- Automated tooling
- Load or stress testing

Unauthorized testing may be treated as hostile activity.

---

## No Bug Bounty

We do not operate a bug bounty program.

Submission of a report:
- Does not grant testing authorization
- Does not guarantee compensation
- Does not provide legal safe harbor beyond good-faith disclosure

---

## Enforcement

We reserve the right to take appropriate action, including referral to law enforcement, for malicious or abusive activity.

Good-faith disclosure is appreciated.  
Unauthorized testing is not permitted.
